Overview

i made a (this) portfolio website to showcase my projects. its pretty basic, using mainly html and css, although i might switch over the react at some point. its got software projects, hardware, and even a bunch of art ive worked on. my main goal with this site was to keep it simple and easy to navigate, and i hope ive done that

Development

development wasnt too bad. a lot of the heavy lifting was taken off my shoulders through the use of a template, so the initial version of the website looked very similar to it. again, it uses html and css since its a static website, and ive gone in and changed a bunch of it to meet my needs. i used github pages to deploy the website and im hosting it with cloudflare. luckily my name isnt very common so i was able to snag a pretty cheap domain name. over time ive made pretty signifigant changes to the site, and since its static and without javascript theres unfortunately been a lot of "manual labor" involved. im still pretty pleased with the website as a whole, though

Reflection

im really glad i was able to get this site together and have a place 100% customizeable to simply display my works. no likes, no followers, no bullshit. its like: heres my project. thats it. in the future though id love to modularize a lot of this stuff and make the workflow for updating the site better, especially since im editing raw html here and would love to be editing a google doc or text file instead

fun fact this was actually a project for Headstarter and I've been using it since, thanks Yasin
