Overview

reminisce was our group's project for Hack Rutgers 2024. its an app meant for those with alzheimers to help them remember their daily activies. it tracks user location and photos and helps build a timeline of their day to day activies, to help prevent memory loss. this was my first solo hackathon where i didnt know anyone, so i was pretty nervous going into this. luckily i found a team of people just looking to have fun and not do anything too serious and we were able to put this together.

Development

to start we used figma to create mockups of the app and how it would function. once we got an understanding of what we wanted out of the app we began development. we split it into ui/ux (frontend) and database and api routing (backend). for the ui we used react native which also helped with location data, and we used mongodb to store info about users, location, and metadata from photos. we linked the database together with javascript and ngrok for api routes and sent it all back to the main application
